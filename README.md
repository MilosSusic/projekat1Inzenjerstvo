# C# Engineering Project

## Overview
This project is focused on demonstrating engineering principles through the use of C#. It aims to cover various aspects of software development, including object-oriented programming, design patterns, and best practices.

## Features
- Implement core engineering concepts
- Utilize object-oriented programming principles
- Include extensive documentation
- Provide examples and use cases

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- .NET Core SDK

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/MilosSusic/projekat1Inzenjerstvo.git
   ```
2. Navigate to the project directory
   ```sh
   cd projekat1Inzenjerstvo
   ```
3. Run the application
   ```sh
   dotnet run
   ```
